# NutriGuide macOS 安裝說明

---

## 只需 3 個步驟

### 第一步：解壓縮

雙擊 `NutriGuide-源碼.zip`，解壓縮到任意位置（例如桌面）。

---

### 第二步：執行安裝程式

1. 按 **Command ⌘ + 空白鍵**，輸入「**終端機**」，按 Enter 開啟
2. 在終端機中輸入 `bash ` （bash 後面加**一個空格**，先不要按 Enter）
3. 把解壓縮資料夾中的 **`安裝NutriGuide.command`** 檔案**拖曳**進終端機視窗
4. 按 **Enter**

終端機會自動完成所有安裝，**不需要其他操作**，等待約 5–10 分鐘即可。

> 第一次執行後，之後可以**直接雙擊** `安裝NutriGuide.command` 執行（腳本會自動幫自己設定好權限）。

---

### 第三步：安裝應用程式

完成後資料夾會自動開啟，找到 **`.dmg`** 檔案：

1. 雙擊 `.dmg`
2. 把 **NutriGuide** 圖示拖曳到 **Applications（應用程式）**
3. 完成！往後直接從應用程式資料夾開啟

> **首次開啟提示：** 若 macOS 顯示「無法驗證」，請**按住 Control 鍵**再點擊 NutriGuide → 選「開啟」→「開啟」，之後就正常了。

---

## 遇到問題？

**終端機顯示 `xcode-select` 相關錯誤**

貼上以下指令安裝 Xcode 工具後重試：

```
xcode-select --install
```
